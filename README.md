# Software-FJ
Sistema Integral de Gestión de Clientes, Servicios y Reservas.
Este sistema es capaz de registrar usuarios nuevos con nombre completo, correo, usuario y password, datos necesarios para tener un login exitoso ingresando el usuario y contraseñas registrados.
Una vez dentro de Software-FJ, podrá adquirir un servicio o una reserva la cual podra cancelar si lo desea, podrá visualizar los detalles de sus actividades, actualiar el correo y ver la información personal.
